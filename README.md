# 🚀 EVI Care Landing Page

A modern, responsive, and high-converting landing page designed for **EVI Care**. Built using HTML, CSS, and JavaScript with a clean UI and smooth user experience.

## ✨ Features

- 📱 Fully Responsive Design
- 🎨 Modern & Professional UI
- ⚡ Fast Loading
- 🖱️ Smooth Scrolling & Animations
- 📞 Book Strategy Call Button
- 💬 WhatsApp Contact Integration
- 📋 Contact Form
- 🔗 Social Media Links
- 🌐 Cross-Browser Compatible

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript

```bash

```
